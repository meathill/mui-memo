## 为什么会有 MuiMemo

MuiMemo 想解决的不是“怎么再多记一点东西”，而是“手正忙的时候，怎么别把那句刚冒出来的小事弄丢”。

很多待办应用默认你有空低头填标题、时间、标签、优先级。但真实日常里，人往往正在走路、收银、带娃、切客户项目，脑子里只有一句大白话。MuiMemo 的目标，是把这句大白话直接收成可执行的任务，而不是先逼你做一次整理。

## 当前阶段

- Web 版已经可用：注册后即可录音、生成任务、搜索、查看输入记录、删除任务。
- 现在的重点在 iOS App：把 Siri 入口、系统提醒、更加稳定的移动端体验做扎实。
- Web 不会被继续往“全能项目管理系统”上堆。它更像一个先把语音意图跑通的产品基线。

## 适合谁

- 手上在忙，不想停下来填字段的人。
- 需要把零碎念头快速收住的人。
- 希望清单更贴近“此刻场景”，而不是像仓库一样平铺的人。

## 不刻意追求什么

- 不在 Web 上硬补不可靠的准时提醒。
- 不把 UI 做成厚重复杂的协作系统。
- 不为了“功能齐全”而把语音入口做得别扭。

## 接下来会继续做的事

- 把“说一句话 -> 形成可执行待办”这条链路做得更稳。
- 在 iOS 上补齐更可靠的系统提醒与更顺手的入口。
- 持续减少录入动作，而不是增加配置项。
