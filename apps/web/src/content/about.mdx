## 为什么会有叨叨记

叨叨记想解决的不是"再多一个待办应用"，而是"手正忙的时候、脑子里冒出来一句话——怎么不让它丢"。

很多 todo 工具默认你有空低头分类、加标签、调时间。但真实日常里你正在走路、收银、带娃、切客户项目，脑子里只有一句大白话。叨叨记的目标是：把这句大白话直接收成可执行的任务，而且**整个 todo 的生命周期**都能继续用语音走完——改时间、勾完成、找回来、再按场景和标签筛出来。

## AI 不只在录入这一步

很多语音 todo 把 AI 用在"录音 → 文字"那一刀，剩下的还是手点。叨叨记不是。

- 说一句新事 → 自动建一条
- 说"改到下午三点" → 改时间
- 说"物业费搞定了" → 自动勾掉
- 说"上次老张那事" → 模糊指代也能找回来
- 说"打电话给老妈，再交物业费" → 一句两件，自动拆

AI 接管的是这条「说 → 想 → 做」的链路，不只是某个节点。

- **后续能力**：系统提醒、Siri 快捷入口、桌面 / 手表这些原生入口会慢慢补；所有新增能力都会先保证不把清单变重。

## 适合谁

- 手上在忙、不想停下来填字段的人
- 需要把零碎念头快速收住的人
- 嫌 Todoist / 滴答清单太重的人
- 想先用手机说一句记下，之后再回到电脑慢慢整理的人

## 不刻意追求什么

- 不在 Web 上硬补不可靠的准时提醒
- 不把 UI 做成厚重复杂的协作系统（即使做协作，也保持轻）
- 不为了"功能齐全"而把语音入口做得别扭
