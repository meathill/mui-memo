# MuiMemo · Design Context

## Users
手忙脚乱的都市年轻人。通勤路上、开会间隙、下厨中途冒出一个念头——没空、也不想打开 app 去戳一排字段填表。他们信任语音，讨厌「整理」。

## Brand Personality
- 温润 · 克制 · 克制
- 像一本薄薄的纸质手账，而不像一块 SaaS 仪表盘
- 3 词：paper, deliberate, calm
- 应该让人想起：一本装帧考究的中文诗集、一张手写的便签、一页编辑部的 colophon
- 不应像：典型硅谷创业公司 landing（紫蓝渐变、卡片网格、英雄大数字、cyan-on-dark）

## Aesthetic Direction
- **主题**：仅 light（已有 paper 色 `#f4ede0`）。Night / Mono 是应用内偏好，不是 landing 主题
- **字体**：Noto Serif SC 做标题 display（中文衬线、温润手账感）；Inter 做正文；JetBrains Mono 专门给编号、标签、页码感元素
- **色**：paper-bg / ink / accent-warm 全部复用现有 token，严守 60-30-10：纸面 60%、墨字 30%、暖橙 accent 10%
- **排版**：左对齐为主，刻意不对称；章节编号像 zine 目录（01 / 02 / 03）；rule 线只画 1px；避免卡片网格
- **留白**：大。fluid spacing 用 clamp。hero 上下呼吸要足

## Design Principles
1. **像手账，不像产品页**。装帧感、页码感、分割线，不是 feature card grid。
2. **序号引路**。章节前冠 `01 / 02 / 03` mono 编号，目录式节奏。
3. **中文优先的 display typography**。Noto Serif SC 不是陪衬，是主角。
4. **暖橙只用在必要处**。CTA、accent marker、单个强调词；其余地方用 ink 不同透明度。
5. **拒绝 AI slop**：不用 left-border stripe、不用 gradient text、不用 purple-cyan、不用「三栏卡片 + 图标 + 标题 + 副文」模板。
